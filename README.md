# 目的
普段discord上で行なっているwikipediaゲームを簡単に行えるようにしたかった

## 構成
**フロントエンド**: TypeScript, React, Recoil

**バックエンド**: Python, Flask

**インフラ, DB**:

    front-end: Firebase Hosting, Github Actions
  
    backend: Cloud Run, Cloud Build, Container Registry
    
    DB: Firestore, Realtime Database
